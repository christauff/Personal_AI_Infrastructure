## CRITICAL ALERTS

### 1. CVE-2026-25536: MCP TypeScript SDK Cross-Client Data Leak
- **Published:** Feb 4, 2026
- **CVSS:** 7.1 (HIGH)
- **Affected:** MCP TS SDK v1.10.0 through v1.25.3
- **Fixed:** v1.26.0 (released Feb 4)
- **Impact:** Race condition in shared StreamableHTTPServerTransport causes response data from one client to leak to another
- **Advisory:** GHSA-345p-7cg4-v4c7
- **PAI Action:** Verify we're using >= v1.26.0 if running any MCP servers with shared transport

### 2. MCP TypeScript SDK v2 Pre-Alpha on Main Branch
- **Status:** Pre-alpha, NOT released
- **Impact:** Massive breaking changes (package split, API rewrite, error hierarchy)
- **Details:** See `breaking-changes/2026-02-05-mcp-sdk-v2.md`
- **PAI Action:** No immediate action. Monitor for RC announcement.
