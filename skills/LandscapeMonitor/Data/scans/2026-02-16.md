## ALERTS

### CRITICAL

**CVE-2026-22778 — vLLM RCE via Video Processing (CVSS 9.8)**
- Affected: vLLM 0.8.3 through 0.14.0
- Two-stage chain: PIL error messages leak heap addresses (ASLR bypass) + heap overflow in OpenCV/FFmpeg JPEG2000 decoder via crafted video frames
- Pre-auth exploitation possible. Full server takeover, model/data access, lateral movement in GPU clusters
- Fix: Upgrade to vLLM 0.14.1+. Only multimodal/video deployments affected.

**CVE-2026-25049 — n8n RCE (CVSS 9.4)**
- Workflow automation platform remote code execution

**GRP-Obliteration — One-Prompt Alignment Destruction (Microsoft, Feb 9)**
- Single unlabeled prompt fine-tunes away safety alignment using inverted GRPO reward signals
- GPT-OSS-20B attack success rate: 13% → 93%. Tested across 15 models, 6 families.
- Tools used for alignment can be weaponized when reward signals are inverted.
